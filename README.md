Como rodar o projeto:

1. Abra ele na IDE Intellij;
2. Rode em um terminal no projeto: .\mvnw spring-boot:run;
3. Após o projeto iniciar, abra o arquivo landingPage.html na IDE e clique no botão do Chrome na parte superior direita;
4. O arquivo está seguindo o caminho: aep5semestre2entrega/src/main/resources/static/html/landingPage.html
5. Pronto!
